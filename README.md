## 프로젝트 소개
아마촌은 **아마추어 개발자들을 위한 프로젝트 매칭 서비스** 입니다. 

</br>

### 기획하게 된 이유
### 😂 프로젝트를 같이 할 팀원을 찾기가 왜 힘들까?
![image](https://github.com/Project-AmaChon/Server/assets/81271328/21545bce-3a3a-4258-b329-4e1487dacd68)

</br></br>

### 🧐 난 이런 사람들을 원해!

<br>

![image](https://github.com/Project-AmaChon/Server/assets/81271328/22b331b6-ad3e-4b25-9cc0-2f0e8ff7900f)

<br>

- 프로젝트 경험을 하고 싶어 같이 할 팀원을 찾아봤지만 찾기 쉽지 않았고, 개발 동아리같은 경우 프로젝트 경험이 있거나 실력있는 사람을 뽑는 경우가 많았습니다. 
- 같은 학교 학생을 대상으로 조사한 결과 비슷한 이유로 프로젝트를 접하기 힘들다 라는 의견이 가장 많았습니다.

<br>

이제 막 시작한 아마추어 개발자에게 프로젝트를 쉽게 접할 수 있는 기회가 많이 없기 때문이라고 생각했고, 이 문제를 직접 해결하고자 '아마촌'프로젝트를 기획하게 되었습니다.

<br>

## 아마촌의 차별점


- **지역 태그를 통해 가까운 위치에 있는 사람들과 프로젝트를 진행할 수 있습니다**
  - 프로젝트를 비대면으로 진행할 수도 있지만, 개인적으로 대면으로 진행했을 때 진행속도도 빨랐을 뿐더러 프로젝트가 흐지부지 되는 경우가 적었습니다
    
- **프로젝트에 적합한 팀원을 추천받을 수 있습니다**
  - 지역 태그와 기술 태그를 기반으로 적합한 팀원을 추천받을 수 있습니다

</br>

## 기능

<br>

<details>
<summary><b>프로젝트 생성</b></summary>
<div markdown="1">


- 프로젝트 생성 시 **기술 태그, 모집 인원, 마감일, 시작 예정일, 제목, 내용**을 설정하여 게시글을 생성할 수 있음
  
- 지역 태그는 **게시글 작성자의 지역 태그로 설정**됨 
 
![image](https://github.com/Project-AmaChon/Server/assets/81271328/b60115bc-2bef-47d3-8d1f-b858bbae0df8) ![image](https://github.com/Project-AmaChon/Server/assets/81271328/c14cbc2d-2de6-4706-8328-ba3b309a3fc2)



</div>
</details>

<details>
<summary><b>프로젝트 필터링 조회</b></summary>
<div markdown="1">


  - 프로젝트 필터링 조회는 지역 태그, 기술 태그, 프로젝트 제목으로 상세하게 필터링하여 조회할 수 있음
    
  - **각 태그는 계층 구조를 가지고 있음**
    - ex. **백엔드 태그**(Spring, Nodejs 등의 백엔드에 속하는 기술 태그를 모두 포함)
    - ex. **용인시 태그**(수지구, 마북구 등을 용인시에 속한 지역 태그를 모두 포함)
  
![image](https://github.com/Project-AmaChon/Server/assets/81271328/86937b4a-c1a1-461d-9d37-a2e5977af695)

</div>
</details>


<details>
<summary><b>프로젝트 상세 조회</b></summary>
<div markdown="1">

  
- 일반 유저의 경우 프로젝트에 **참가 신청**을 할 수 있음

- 게시글 작성자의 경우 **프로젝트 참가 신청 현황 조회와 팀원 추천**을 이용할 수 있음

![image](https://github.com/Project-AmaChon/Server/assets/81271328/2b68db67-6362-4b14-9521-cfea8fb80ef4) ![image](https://github.com/Project-AmaChon/Server/assets/81271328/e3810500-ab12-4772-989d-fa38685ad021)


</div>
</details>

<details>
<summary><b>프로젝트 신청 현황 조회</b></summary>
<div markdown="1">


- 게시글 작성자는 프로젝트 상세 조회 탭에서 신청 현황을 조회할 수 있고, 수락 또는 거절을 할 수 있음

- 이미 거절된 사람은 다시 신청할 수 없으며, 신청이 수락된 사람은 팀원(참가 중인 인원)으로 속하게 됨


![image](https://github.com/Project-AmaChon/Server/assets/81271328/d1114a16-f23c-46fc-b892-b6e9770bab84) ![image](https://github.com/Project-AmaChon/Server/assets/81271328/c8005130-11b9-4169-8ab6-9103bfcdd2ad)

</div>
</details>

<details>
<summary><b>프로젝트 팀원 추천</b></summary>
<div markdown="1">


- 프로젝트의 **기술 태그와 지역 태그를 기반으로 추천**이 이루어짐

- 기술 태그가 일치하면 각 태그당 +4, 지역 태그가 일치하면 +5 점을 부여

- 점수가 가장 높은 순으로 추천이 이루어짐

- 더 디벨롭 된다면 다양한 요인을 고려하여 더 적합하게 추천할 예정

![image](https://github.com/Project-AmaChon/Server/assets/81271328/a1a8ee65-9890-44b2-9115-82e4a8189dac) ![image](https://github.com/Project-AmaChon/Server/assets/81271328/dd10e42b-f43f-406e-9135-fa1c3488c882)


</div>
</details>

<summary><b>마이페이지(프로필) 조회</b></summary>
